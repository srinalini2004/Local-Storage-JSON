# L1 - Enhanced Todo List with localStorage and Search

## Features
- Add new tasks
- Mark tasks as completed (click the task text)
- Delete tasks
- Search tasks in real-time
- All tasks are saved in localStorage as a JSON array
- Each task contains:
  - Unique ID
  - Task text
  - Completion status

## How to Use
1. Open the application in your browser.
2. Type a task in the input field and click **Add Task**.
3. Click on any task to mark it as completed.
4. Click the red **X** button to delete a task.
5. Use the search bar to filter tasks instantly.
6. All tasks remain saved even after refreshing the page.

## Tech Used
- HTML
- CSS
- JavaScript
- localStorage API
